
## 注释规范

#### 注释以字符 /* 开始，以字符 */ 结束 
#### 注释不能嵌套

#### 单行注释

推荐：

/* Comment Text */
.jdc{}

不推荐：
.jdc{
	display: block;/*Comment Text*/
}

#### 模块注释

注释内容第一个字符和最后一个字符都是一个空格字符，/* 与 模块信息描述占一行，多个横线分隔符-与*/占一行，行与行之间相隔两行

推荐：

/* Module A
---------------------------------------------------------------- */
.mod_a {}

不推荐：

/* Module A ---------------------------------------------------- */
.mod_a {}
/* Module B ---------------------------------------------------- */
.mod_b {}

#### 文件信息注释

@charset "UTF-8";
/**
 * @desc File Info
 * @author Author Name
 * @date 2015-10-10
 */
